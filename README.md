# Nested Cross-Validated Double Machine Learning for PLIV

Replication repository for the paper

**Nested Cross-Validated Double Machine Learning for the Partially Linear IV Model**  
Tamer Çetin

Submitted to **The Econometrics Journal**

---

## Repository structure

src/
Implementation of Nested-CV DML-IV estimator

notebooks/
Monte Carlo and empirical notebooks

simulations/
Scripts used to generate simulation results

results/
Tables and figures produced by the simulations

replication/
Scripts that reproduce the entire paper

---

## Requirements

Python 3.10+

Install dependencies

pip install -r requirements.txt
